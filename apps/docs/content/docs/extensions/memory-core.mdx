# Memory (File) Extension

The Core Memory extension provides basic, file-backed long-term storage for user conversations. It uses simple JSON storage to remember context across restarts.

## Setup Guide

Because it relies purely on local Node.js `fs` file modifications, setup does not require any passwords or external keys!

1. Go to the Superclae Dashboard and locate the **Memory** category.
2. Click **Connect** on the Memory (File) card.
3. You will be prompted to enter a **Storage Path**.
   - The default is usually `./data/memory`.
   - You can enter an absolute server path if you prefer storing the JSON brain files in a custom volume or partition.
4. Click Connect!

Your AI agents will immediately begin saving critical context blocks and returning conversation chunks persistently based on user IDs.
