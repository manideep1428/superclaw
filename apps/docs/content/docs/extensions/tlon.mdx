# Tlon / Urbit Extension

Connect your SuperClaw AI as a "Ship" on the decentralized, peer-to-peer Urbit OS network (accessible via clients like Tlon).

## Setup Guide

Urbit requires a Ship URL and an Access Code to authenticate clients securely over HTTP.

### 1. Get the Web Login Code
1. Open your Urbit terminal (the `dojo`). This could be a local process or a hosted planet (e.g. on NativePlanet, Tlon hosting, or Red Horizon).
2. Into the `~your-planet-name:dojo>` prompt, type:
   ```bash
   +code
   ```
3. Press Enter. It will spit out a 4-word, hyphenated code like `lidlut-tabwed-pillex-ridrup`.
4. Copy this string entirely. This is your Ship's **Access Code**.

### 2. Determine your Ship URL
- If hosting on a VPS or Provider, this is your domain name (e.g. `https://your-planet.tlon.network`).
- If hosting locally, it is usually `http://localhost:8080`.

### 3. Connect to SuperClaw
1. In the SuperClaw dashboard, click **Connect** on the Tlon / Urbit card.
2. Enter your **Ship URL**.
3. Paste the **Access Code** from the Dojo.
4. Click Connect! Your AI can now sit in Landscape groups and chat peer-to-peer!
