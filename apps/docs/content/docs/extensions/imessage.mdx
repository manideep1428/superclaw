# iMessage (Native) Extension

The easiest way to integrate with iMessage. This extension uses native macOS AppleScript and SQLite DB reads to interface with the built-in Messages app on your Mac.

**Crucial Requirement**: Superclae must be running locally on a macOS machine with the Messages app logged into an Apple ID.

## Setup Guide

There is **zero configuration required** in the Superclae dashboard.

1. Open Superclae on your Mac.
2. Go to the Extensions page.
3. Click **Connect** on the iMessage (Native) card.
4. Your Mac may prompt you asking "Terminal / node would like to control the Messages app". You **must click OK / Allow** for AppleScript permissions to be granted.
5. You're done! Your AI will now intercept incoming iMessages and reply using your Mac's active session.
