## LanceDB Vector Memory

The Superclae LanceDB extension handles true long-term memory for your AI. By default, it uses an OpenAI embeddings model (`text-embedding-3-small`) to convert text into mathematical vectors for semantic search.

### Getting your Embedding API Key

1. Go to the [OpenAI Developer Platform](https://platform.openai.com/api-keys).
2. Sign in or create an account.
3. On the left sidebar, click **API Keys**.
4. Click **Create new secret key**.
5. Give it a name like "Superclae Embeddings" and click **Create secret key**.
6. **Copy the key securely**. It will look like `sk-...`.

Paste this into the **OpenAI API Key (for embeddings)** field in Superclae.

*(If you are self-hosting your embeddings API via Ollama/vLLM, you can provide an arbitrary string here as long as Superclae is configured to hit your local endpoint).*
