# LINE Messaging API

The LINE extension enables your SuperClaw agent to respond to messages from users on LINE.

## Requirements

1. LINE Business ID
2. Channel Access Token
3. Channel Secret

## How to get credentials

1. Go to the [LINE Developers Console](https://developers.line.biz/console/).
2. Create a **Provider** if you don't have one (this represents your organization).
3. Inside your provider, click **Create a new channel** and select **Messaging API**.
4. Fill in your bot's details (Name, description, icon) and create it.
5. Once created, go to the **Basic Settings** tab. Scroll down to find the **Channel secret** and copy it into SuperClaw.
6. Switch to the **Messaging API** tab. Scroll to the bottom and click **Issue** under **Channel access token (long-lived)**. Copy this token into SuperClaw.
7. Also on the **Messaging API** tab, locate the **Webhook URL** field. 
8. Take your SuperClaw URL (e.g., `https://my-claw.foo.com/api/webhooks/line`) and enter it here, then click **Verify**. Note: Make sure to **Enable** the webhook use toggle!
