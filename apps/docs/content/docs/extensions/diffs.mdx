# Diff Viewer Extension

The Diff Viewer extension allows your AI to automatically render complex code `diffs` directly into visual formats (HTML webpages, raw PNG images, or generated PDFs).

This is incredibly useful if you connect your agent to messaging platforms that do not support raw markdown blocks well, or if you want it to output beautiful visual code reviews via email/Slack.

## Setup Guide

There is **zero configuration required** in the Superclae dashboard! The Diff Viewer tooling utilizes lightweight local rendering pipelines built directly into the openclaw core.

1. Go to the Superclae Dashboard and click the **Tools** category.
2. Click **Connect** on the Diff Viewer card.
3. Done! 

When your AI generates a patch diff for code modifications, it will automatically leverage the viewer to wrap and format the output visually if requested by the user.
