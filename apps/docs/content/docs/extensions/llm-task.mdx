# LLM Task Tool Extension

The LLM Task extension provides your AI agents with a deterministic, structured JSON-generation environment. If your conversational agent needs a sub-process to strictly extract JSON data from user input without breaking character or hallucinating formats, it hands the request off to the LLM Task pipeline!

## Setup Guide

There is no external SAAS to authenticate. It proxies through your connected Providers.

1. Go to the SuperClaw Dashboard and find the **LLM Task** extension under the Tools category.
2. Click **Connect**.
3. Set your preferred **Default Model** (e.g. `gpt-4o` or `claude-3-opus-20240229`). 
   *(Note: You must have the corresponding AI Provider enabled in your dashboard so the LLM Task tool can dispatch inference requests!)*
4. Set optional Advanced Settings like **Max Tokens** and **Timeout (ms)** limits.
5. Click Connect!

Your AI now has access to the `/task` internal route!
