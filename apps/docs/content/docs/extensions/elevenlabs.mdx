## Talk Voice (ElevenLabs TTS)

ElevenLabs provides hyper-realistic text-to-speech for Superclae.

### Getting your API Key

1. Go to [ElevenLabs](https://elevenlabs.io/) and create an account or sign in.
2. In the bottom left corner, click your profile picture and select **Profile + API key**.
3. To view your API Key, click the eye icon to reveal it.
4. Copy the API key string.

Paste this into the **API Key** field in the Superclae Talk Voice extension configuration! Note that ElevenLabs charges credits per character generated.
