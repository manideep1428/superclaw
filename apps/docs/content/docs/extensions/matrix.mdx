# Matrix Extension

Matrix is a decentralized and encrypted chat protocol used in various self-hosted platforms like Element.

## Configuration Guide

1. **Homeserver URL**: The full URL to your Matrix server. E.g. `https://matrix.org` or `https://matrix.your-domain.com`.
2. **Bot User ID**: This is the full Matrix handle of your bot, starting with an `@` and ending with the domain. E.g. `@my-bot:matrix.org`.
3. **Access Token**: This requires you to create your bot user in your homeserver, login to element, and run the `/token` command or use a dedicated script to acquire a permanent token.

To get an access token for an established user on element:
1. Log in to the bot's account on **Element Web**.
2. Open Settings -> **Help & About**.
3. Under the Advanced section, scroll down and find the **Access Token**. Paste this into SuperClaw!
