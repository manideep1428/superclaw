# Extensions in Superclae

Superclae is built around a plug-and-play architecture. Extensions are the modules that connect your core AI brain to the outside world.

## How to use this documentation

If you want to connect an extension (for example, you want your agent to reply on Slack), simply click **Slack** on the left sidebar.

Each extension page provides a step-by-step walkthrough on:
- Where to register your app on the target platform.
- Which API keys, Tokens, and Webhook Secrets you need to copy.
- Where exactly to paste your Superclae webhook endpoint so they can communicate.

Click an extension on the left sidebar to get started!
