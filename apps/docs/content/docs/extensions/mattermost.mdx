# Mattermost Extension

Connect your Superclae AI directly to your self-hosted or cloud Mattermost enterprise workspaces.

## Setup Guide

To get your Mattermost Bot Token so you can converse with your AI in Mattermost:

### 1. Enable Bot Accounts in Mattermost
If you are the System Admin (or ask yours):
1. Go to **System Console** > **Integrations** > **Bot Accounts**.
2. Set **Enable Bot Account Creation** to `true`.

### 2. Create the Bot Account
1. From the normal Mattermost product menu (top left corner of the channel view), select **Integrations** -> **Bot Accounts**.
2. Click **Add Bot Account**.
3. Fill in the details:
   - **Username**: Must begin with a letter, e.g., `Superclae-bot`.
   - **Role**: `Member` generally works, or `System Admin` if you want it to have unrestricted access to all channels.
4. Check any extra permissions like "post to all channels" if needed.
5. Click **Create Bot Account**.

### 3. Save the Token
Once created, Mattermost will display a "Setup Successful" page with an **Access Token**.
**Copy this token immediately**. You will not be able to see it again!

### 4. Connect to Superclae
1. Open the Superclae Dashboard and click **Connect** on the Mattermost card.
2. Enter your **Mattermost Server URL** (e.g. `https://chat.mycompany.com`).
3. Paste the **Bot Access Token** you copied earlier.
4. Click Connect!

Your AI bot is now live inside Mattermost. You can add it to channels and mention it to start conversing!
