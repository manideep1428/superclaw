# Twitch Chat

Interact with viewers directly in your Twitch streams!

## Configuration

1. **OAuth Token**: Head over to twitchapps.com/tmi to generate an oauth token for your bot account.
2. **Bot Username**: This is the username of your Twitch bot. E.g. `superclaw_bot`. The OAuth token should belong to this user.
3. **Channel Name**: This is your main streaming channel, or wherever you want the bot to sit and listen to chat.

## Setting Up

1. Log into Twitch as the bot you created.
2. Go to [twitchapps.com/tmi](https://twitchapps.com/tmi/) and authorize the application. This will give you an OAuth token.
3. Keep the "oauth:" prefix on the token when pasting it into SuperClaw.
4. Set the bot username and channel in SuperClaw, then click **Connect**.
