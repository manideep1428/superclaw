# IRC Extension

This extension allows your AI to connect to IRC servers, join channels, and handle private messages. 

## Setup Guide

To connect your bot to an IRC network, you just need standard IRC connection details.

1. **IRC Server**: The address of the server (e.g., `irc.libera.chat`).
2. **Port**: Usually `6697` for SSL connections, or `6667` for unencrypted connections.
3. **Bot Nickname**: The username/nick your bot will use (e.g., `Superclae-bot`).
4. **Channels**: A comma-separated list of channels to join upon connecting (e.g., `#general, #support`).
5. **Server Password**: (Optional) If the server requires a password or if you are using NickServ identification via SASL/Server pass.

When you click **Connect** on the Superclae dashboard, the bot will immediately attach to the IRC network and join the configured channels!
